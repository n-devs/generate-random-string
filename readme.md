# generate-random-string

> generate random string

## Install

```bash
npm i -S generate-random-string

or

yarn add generate-random-string
```

## Usage

```javascript
const generateRandomString = require('generate-random-string');

const uid = generateRandomString(20);
		// console.log(uid);
		//=>0kgStW1N1QDJi0IOfmhQ
```
